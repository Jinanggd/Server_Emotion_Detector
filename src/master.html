Master Emotion Detector

Está página consiste en visualizar los datos de duración de las emociones que se encuentran guardados en la base de datos de REDIS

Para visualizarlos el usuario deberá de:

1) Introducir el ID de almacenamiento ( por defecto es "ED" )

2) Darle al botón de visualizar

3) Se desplegará un listado JSONS en formato string ( Datos )

4) Para visualizarlos, utilize un visualizador de JSONS como por ejemplo JSON Converter
